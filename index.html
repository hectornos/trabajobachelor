Escoge cualquier tema para empezar...

Unidad uno

En este tema hay ejercicios sencillos para entender el funcionamiento básico de Javascript.

Unidad dos

Continuamos aprendiendo JS en esta unidad profundizaremos en la estructura del DOM de un documento html y en cómo podemos modificarlo mediante el uso de JavaScript.

Unidad tres

Un lenguaje como Javascript no solamente nos permite añadir funcionalidad a nuestra página si no que también crear funciones transparentes para el usuario pero muy utiles como las validaciones de formularios...

Unidad cuatro

En esta unidad aprenderemos a utilizar Ajax que nos permite recargar solamente una parte de cada página, ganando en velocidad. Por otra parte el uso de JQuery simplifica y acorta las funciones de Javascript clásicas.

Unidad cinco

Por último aprenderemos estilos y cómo se pueden aplicar a nuestra página hecha con HTML5 mediante el uso de plantillas css3, con el fin de dar un aspecto diferente a la página que le permita destacar.
